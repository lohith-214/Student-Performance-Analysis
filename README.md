# Student Performance Analysis

This project analyzes student academic performance using Python.  
It explores subject-wise averages and the relationship between attendance and marks.

## Dataset
- `students.csv` contains students' marks and attendance data.

## Features
- Handles missing values
- Computes average marks for each subject
- Analyzes the impact of attendance on performance
- Visualizes results using Matplotlib

## Technologies Used
- Python
- Pandas
- Matplotlib

## Project Structure
- `analysis.py` – Main analysis and visualization script
- `students.csv` – Dataset used for analysis
- `README.md` – Project documentation

## Key Insights
- English has the highest average score
- Math shows comparatively lower performance
- Higher attendance correlates with better academic results

## How to Run
1. Install dependencies:
   pip install pandas matplotlib
2. Run the script:
   python analysis.py
